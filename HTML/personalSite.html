The Dawg House

The Dawg House is a project created by myself and three others as a final project for INFO 343 (client side web development). The idea was to create a service similar to yelp, but for housing in the University District. On the site, users are able to view apartments, read reviews, and create reviews, which are updated on the map in real time. My role in the project was to write the JavaScript to create the map and place the markers on it. I also wrote the PHP for users to submit reviews, and to write the Latitude and Longitutde of the street addresses to our JSON object.
